# Forecast-elicitation-Mechanism

See also [Femtools](https://github.com/Hermera/Forecast-elicitation-Mechanism).

The goal of this project is to implement these papers:

* [A Bayesian truth serum for subjective data](https://economics.mit.edu/files/1966) (BTS)
* [Water from Two Rocks: Maximizing the Mutual Information](https://arxiv.org/abs/1802.08887) (MCG)
* [Dominantly Truthful Multi-task Peer Prediction with a Constant Number of Tasks](https://arxiv.org/abs/1911.00272) (DMI)
* [Informed Truthfulness in Multi-Task Peer Prediction](https://arxiv.org/abs/1603.03151) (CA)
